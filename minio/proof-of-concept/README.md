# MinIO Proof of Concept

Простой скрипт для демонстрации загрузки и выгрузки файлов в S3 хранилище с помощью
pre-signed POST и GET.

```bash
# В .env можно указать параметры для подключения к S3 хранилищу
cp .env.example .env

# Желательно создать и активировать виртуальное окружение
# python -m venv .venv && . .venv/bin/activate
pip install -r requirements.txt

# Скрипт загрузит тестовую картинку в S3 хранилище, а затем выгрузит её обратно,
# имитируя взаимодействие фронтенда и бэкенда 
python main.py
```
